    After Web Development, try learning Python or Lua, but first try Lua since its an easier programming language and its a smaller version of Python, both have a huge amount of range from Front End to Back End Development.

    And then try C# for Machine Learning or Computer Science, its like a little brother of C++ and C, but more simplified.
